# Caregiver Support (Demo • Non-Diagnostic) — Streamlit App

A small, **demo-only** caregiver-support app built with **Streamlit**.
- ✅ **No OpenAI / API keys**
- ✅ Uses only **rule-based logic** (regex + keywords)
- ✅ **Non-diagnostic**: not medical advice, no diagnosis, no medication changes
- ✅ Deployable on **Streamlit Community Cloud (free)**

## Features
- **Care Note Builder**: generates structured notes you can copy/paste
- **Support Suggestions**: keyword-based practical suggestions and safety nudges
- **Quick Check-in**: caregiver-focused next steps (non-medical)
